# <img width="32" height="32" alt="AGYD_logo" src="https://github.com/user-attachments/assets/b26aaccc-536b-46b9-b87f-970150b5d91c" /> AGYD-AnotherGenericYoutubeDownloader—
Implementing a GUI and some enhancements for using the yt_dlp package.

## With AGYD you CAN [By the date this is written]:
* Download music.
* Download music playlists (Like Albums).
* Download Videos (By now, 480p on YouTube).

## Supported formats:
* Audio: mp3.
* Video: mp4.
