# <img width="32" height="32" alt="AGYD_logo" src="https://github.com/user-attachments/assets/b26aaccc-536b-46b9-b87f-970150b5d91c" /> AGYD-AnotherGenericYoutubeDownloader—
Implementing a GUI and some enhancements for using the yt_dlp package.

## With AGYD you CAN [By the date this is written]:
* Download music.
* Download music playlists (Like Albums).
* Download Videos (By now, 480p on YouTube).

## Supported formats:
* Audio: mp3.
* Video: mp4.

## FOR USE OF SOURCE CODE
### 1) Clone or download (in this case you have to extract it) the latest committed code.
### 2) You must have the ffmpeg full build (ffmpeg-git-full or similar):
* In the recommended case, use gyan.dev FFmpeg Builds, [Link Here](https://www.gyan.dev/ffmpeg/builds/).
 
<img width="1542" height="469" alt="Screenshot_1" src="https://github.com/user-attachments/assets/43344522-b656-477a-a2c6-b3967013c858" />

* You must have it on the same directory as the AGYD code.

### 3) Install the requirements.txt
> pip install -r requirements.txt

## Dependencies
* yt-dlp [Link to the project](https://github.com/yt-dlp/yt-dlp)
* FFmpeg [Link to the project](https://github.com/FFmpeg/FFmpeg)
