# AGYD-AnotherGenericYoutubeDownloader-
Implementing a GUI and some enchanments for use the yt_dlp package.

## With AGYD you CAN [By the date this is wroten]:
* Download music.
* Download music playlists (Like Albums).
* Download Videos (By now, 480p on Youtube).
